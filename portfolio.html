A weather dashboard web application that allows the user to search for a city and have it's five-day weather forecast be displayed to the screen. Past searches are stored within local storage and displayed on the side of the page. The site retrieves weather forecast data using the OpenWeather API; HTTP requests are implemented with use of AJAX, JSON to get data from third-party server-side API.
