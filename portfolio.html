Date Night Planner

A web application to help users plan their date nights by helping them find a theatre close to their location, a movie playing at that theatre, and a showtime to watch at. A map is displayed to help guide the user to where they should go, and displays their information to the screen. Uses Mapbox' API for geolocation data, and Gracenote's API to generate a list of theatres, the movies playing and their showtimes. Firebase is used for store login data.

Weather Dashboard

A weather dashboard web application that allows the user to search for a city and have it's five-day weather forecast be displayed to the screen. Past searches are stored within local storage and displayed on the side of the page. The site retrieves weather forecast data using the OpenWeather API; HTTP requests are implemented with use of AJAX, JSON to get data from third-party server-side API.

Work Day Planner

A simple work-day planner web application that allows a user to input text to remind themselves of tasks to do at each hour of the work day. Pressing the save button to the side will store the text data to the browser's local storage so it can be viewed at a later time. Built using HTML, CSS, and Javascript.
